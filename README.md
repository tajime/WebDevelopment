# WebDevelopment
Githubの使い方
    ・URL
        https://github.com/tajime/WebDevelopment.git

    ・データを送るとき
        git add 送るファイル :　 (「--all」で全て送る)
        git commit -m "コメント" : 
        git push origin ブランチ名 :(masterはメインブランチ)
        
    ・ブランチの作り方
        git branch ブランチ名 : ブランチを作る
        git branch : 現存するブランチの確認
        git checkout ブランチ名 : 更新するブランチの選択
